# website-blocker
A simple program to stop your unhealthy visits to unproductive websites.

## Installation
* *Python 3.x*
* *Tested on Windows*

## Usage
It works only on Windows.

In order to make it work for Mac and Linux use: `hosts_path = '/etc/hosts'`
